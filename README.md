# Job Application Form

A responsive **Job Application Form** built using **HTML5** and **CSS3**. This project demonstrates the use of CSS Grid to create a clean, modern, and responsive form layout without relying on any CSS frameworks.

## 📸 Preview

A simple job application form that includes:

- First Name
- Last Name
- Email Address
- Job Role Selection
- Address (Textarea)
- City
- Pincode
- Start Date
- CV Upload
- Submit Button

The form automatically adjusts its layout for smaller screen sizes using CSS Grid and media queries.

## 📱 Responsive Design

The form uses:

```css
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
```

This allows the layout to automatically adjust the number of columns based on the available screen width.

On smaller screens, the textarea spans only one column to maintain a clean layout.
